# 🚀 JSON Bulk Importer - from AnkiVN with ❤️

An Anki addon for fast bulk Create/Update cards from JSON with practical import tools.

Website: https://ankivn.com

## Main Features

- Smart Sync by `__guid__` or selected match field.
- Save/Load Preset for repeated workflows.
- Generate missing `__guid__` values.
- Add selected deck into JSON via `__deck__`.
- Table preview with edit mode and HTML preview mode.
- Media field config + pre-fetch for image/audio URLs.
- Batch history records saved as JSON files.
- Bilingual Help dialog (English / Vietnamese).

## Open in Anki

Menu path:

`AnkiVN` -> `🚀 JSON Bulk Importer - from AnkiVN with ❤️`
